# 投放数据管理系统使用手册与部署文档

## 1. 系统概述
本系统是 RESMO PMS 的一个子模块，旨在帮助业务部门高效管理月度投放数据。系统支持 Excel 批量导入、数据查询、多维度统计分析及可视化报表。

## 2. 功能模块

### 2.1 数据看板 (Dashboard)
- **关键指标卡片**: 展示总访客数、总花费、总成交单数、总成交金额。
- **流量趋势图**: 对比自然流量与推广流量的每日变化趋势。
- **花费趋势图**: 堆叠展示曝光花费、天猫进店成本等。
- **转化率趋势图**: 结合柱状图（加购数）和折线图（支付转化率）展示转化效果。

### 2.2 数据查询 (Data List)
- **列表展示**: 详细展示每日的各项投放数据。
- **日期筛选**: 支持按日期范围筛选数据。
- **导出 Excel**: 支持将筛选后的数据导出为 Excel 文件。
- **数据删除**: 管理员权限可删除错误数据。

### 2.3 数据录入 (Data Entry)
- **Excel 导入**: 支持批量上传 Excel 文件。
- **智能解析**: 自动识别第4行开始的数据，支持 UPSERT（更新或插入）逻辑，相同日期的数据会自动覆盖，方便修正历史数据。
- **数据预览**: 导入前提供数据预览，确认无误后保存。

## 3. 部署说明

### 3.1 环境要求
- Node.js >= 18.0.0
- NPM / Yarn

### 3.2 部署步骤
1. **安装依赖**:
   ```bash
   npm install
   ```
   *注意: 本次更新引入了 xlsx, recharts 等库，请确保依赖已安装。*

2. **启动服务**:
   - 开发环境: `npm run dev` (同时启动前端和后端)
   - 生产环境构建:
     ```bash
     npm run build
     npm run server
     ```

3. **数据备份**:
   - 系统数据存储在 `server/data/store.json`。
   - 建议定期备份该文件。
   - 系统内置了"一键备份数据"功能，管理员可在侧边栏底部点击下载完整数据备份。

## 4. 数据格式说明 (Excel 模板)
导入的 Excel 文件需遵循以下列顺序（从第4行开始读取数据）：
1. 日期 (必填, YYYY-MM-DD)
2. 自然流量-访客数
3. 自然流量-加购数
4. 自然流量-支付买家数
5. 推广流量-访客数
6. 推广流量-加购数
7. 推广流量-支付买家数
8. 天猫品牌词-进店数
9. 天猫品牌词-累计加购数
10. 天猫品牌词-累计支付买家数
11. 曝光花费
12. 天猫品牌进店成本
13. 加购率 (小数)
14. 支付转化率 (小数)
15. 直接支付金额
16. 京东搜索词-进店数
17. 京东搜索词-购数
18. 京东搜索词-成交客户数
19. 京东搜索词-成交金额
20. 总访客数
21. 总成本

## 5. 权限说明
- **管理员 (Admin)**: 拥有所有权限，包括录入、修改、删除、备份、恢复。
- **编辑 (Editor)**: 拥有录入、修改权限。
- **查看者 (Viewer)**: 仅拥有查看和导出权限。
